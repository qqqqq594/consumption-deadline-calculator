소비기한 계산기

제조일을 1일 차로 보고 소비기한을 계산하고, 오늘 기준 남은 날짜만으로 제조일을 역산하는 새 탭도 같이 쓸 수 있게 정리했어.

1. 제조일 입력

날짜 선택기는 오늘 날짜를 기본값으로 열어둘게. 수동 입력과 날짜 선택기는 서로 자동으로 맞춰진다.

2. 소비기한 일수 입력

이제는 제조일이 바로 1일 차야. 그래서 제조일 포함 총 X일 상품의 소비기한 마지막 날은 제조일 + (X - 1)일로 계산해.

3. 계산 결과

제조일 -
법적 소비기한 마지막 날 -
법적으로 먹을 수 있는 마지막 시점 -
법적으로 소비기한이 지난 시점 -
최종 소비기한에서 X일 전 날짜 -
오늘 기준 상태 -
개별 제품 표시사항이 우선이고, 실제 표시 방식이나 관련 규정 해석은 제품군별 사정이 있을 수 있다. 이 계산기는 입력한 날짜와 일수를 기준으로 날짜 계산을 돕는 용도다.

4. 소비기한 1일 전부터 7일 전까지

구분 날짜

1. 오늘 기준 고정 상품 제조일 역산

오늘 날짜를 기준으로 남은 날짜를 더해 소비기한 마지막 날을 잡고, 제조일을 1일 차로 포함해 제조일을 역산해.

2. 직접 입력해서 제조일 계산

상품명 없이 숫자 두 개만 넣으면 돼. 남은 날짜는 오늘부터의 순수 날짜로 넣고, 총 소비기한은 제조일 포함 일수로 계산해.

기준일은 항상 오늘이야. 예를 들어 오늘 7일 남은 30일 상품이면 제조일은 자동으로 역산된다.

3. 직접 계산 결과

기준일 -
소비기한 마지막 날 -
제조일 남은 날짜와 총 소비기한을 넣으면 여기서 제조일을 바로 확인할 수 있어.
